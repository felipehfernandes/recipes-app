# Recipes App!

  <summary><strong>👨‍💻 O que deverá foi desenvolvido</strong></summary><br />

  Foi desenvolvido em grupo um app de receitas, utilizando o que há de mais moderno dentro do ecossistema React: Hooks e Context API!

  Nele é possível: ver, buscar, filtrar, favoritar e acompanhar o progresso de preparação de receitas de comidas e bebidas!

  ⚠️ A base de dados é composta de 2 APIs distintas, uma para comidas e outra para bebidas.

  O layout tem como foco dispositivos móveis, dessa forma todos os protótipos vão estar desenvolvidos em telas menores.

  <summary><strong>:memo: Habilidades</strong></summary><br />

  Nesse projeto, eu fui capaz de:

  - Utilizar _Redux_ para gerenciar estado
  - Utilizar a biblioteca _React-Redux_
  - Utilizar a Context API do _React_ para gerenciar estado
  - Utilizar o _React Hook useState_
  - Utilizar o _React Hook useContext_
  - Utilizar o _React Hook useEffect_
  - Criar Hooks customizados
  
  <summary><strong>🛠 Testes</strong></summary><br />
  
  Para o projeto ser validado, foram desenvolvidos testes unitários para cada componente da aplicação

  * <details><summary><b> TheMealDB API</b></summary>

      O [TheMealDB](https://www.themealdb.com/) é um banco de dados aberto, mantido pela comunidade, com receitas e ingredientes de todo o mundo.

      Os end-points são bastante ricos, você pode [vê-los aqui](https://www.themealdb.com/api.php)
      
  * <details><summary><b> The CockTailDB API</b></summary>
      Bem similar (inclusive mantida pela mesma entidade) a TheMealDB API, só que focado em bebidas.

      Os end-points também são bastante ricos, você pode [vê-los aqui](https://www.thecocktaildb.com/api.php)
